## 🌟 Features

- ✅ **50,000+ Pre-trained Conversations** (Hindi + Hinglish + English)
- ✅ **Auto-Learning from Chat** - Gets smarter with every conversation
- ✅ **Natural Indian Chat Style** - Speaks like real Indians
- ✅ **Multiple Response Variations** - Never repeats same reply
- ✅ **Emoji Support** - Modern chat experience
- ✅ **Group & Private Chat** - Works everywhere
- ✅ **Sticker Replies** - Learn and reply with stickers
- ✅ **Admin Controls** - Enable/disable per group
- ✅ **MongoDB Powered** - Fast and scalable

## 🚀 Deploy

### Variables

- `API_ID` - Get from my.telegram.org
- `API_HASH` - Get from my.telegram.org
- `BOT_TOKEN` - Get from @BotFather
- `MONGO_URL` - MongoDB connection string
- `OWNER_ID` - Your Telegram user ID

### Setup

```bash
git clone https://github.com/DAXXTEAM/DAXXCHATBOT
cd DAXXCHATBOT
pip3 install -r requirements.txt
python3 -m nexichat
```

## 📝 Commands

- `/start` - Start the bot
- `/chatbot` - Enable/disable in groups (Admin only)
- `/ping` - Check bot status
- `/id` - Get user/chat ID
- `/stats` - View bot statistics
- `/broadcast` - Broadcast message (Owner only)

## 🎯 How It Works

1. **Pre-trained**: Comes with 50K+ conversations
2. **Auto-learns**: Watches group chats and learns patterns
3. **Smart replies**: Uses learned patterns to respond naturally
4. **Random selection**: Picks from multiple responses for variety

## 💡 Training Tips

The bot automatically learns from conversations:

```
User A: "bhai kaise ho"
User B: "ekdum mast yaar 😎"

Bot learns: "bhai kaise ho" → "ekdum mast yaar 😎"
```

More conversations = Smarter bot!
